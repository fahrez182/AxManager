# Re-ignite（重启系统）

::: info 简而言之
**Re-ignite** 是一个重启插件/模块以应用更新的系统——类似于在手机上重启 Root 模块，但是针对 **AxManager** 的。
:::

[[toc]]

## 背景 — Re-ignite 概念

在 **Root 模块**中，重启是激活或更新模块的重要过程，以便更改能够在系统级别完全生效。然而，这种方法对于**插件**来说并不可行，因为插件完全在**用户空间**内运行，并且**无法访问****启动期间的挂载过程**。

作为一种变通方法，我开发了 **_Re-ignite_** 概念——一种直接_重新初始化_插件的机制，执行多个内部操作来模拟重启的效果，但仅限于插件本身。这允许插件刷新和应用更新，而无需完整的系统重启。
